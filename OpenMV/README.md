# OPENMV Code
Code specific to the OpenMV board

## Requirements

OpenMV IDE

OpenMV4 Board


## Installation

IDE can be downloaded from: https://openmv.io/pages/download

## Relevant Info
STM32H767 Processor

ARM Cortex-M7

All pins except Pin 6 are 5V tolerant. Max I_in is 25mA

3.3V output max I_out is 250 mA
